# Event Registration & Check-In Validation System

MERN app with validated registrations, email verification, and QR-based attendance.

## Quick Start

### Server
```bash
cd server
cp .env.example .env
npm i
npm run dev
```
Set `CLIENT_URL=http://localhost:5173` in `.env`.

### Client
```bash
cd client
npm i
npm run dev
```
Set `VITE_API_URL=http://localhost:4000/api` in `.env` (optional).
